# Warning
Backup sensitive data. Check the content of the files before proceeding. Use at your own risk, manipulating kernel parameters can break it.

# Installation
```bash
git clone https://github.com/MikeHorn-git/Kernel-Hardening.git
cd Kernel-Hardening
chmod +x ./install.sh
sudo ./install.sh
```
